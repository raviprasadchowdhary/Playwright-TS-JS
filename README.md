## ğŸ› Troubleshooting

### Common Issues

**Port already in use:**
- Ensure port 4200 is not being used by another application
- Close any running instances before starting

**Installation errors:**
- Delete `node_modules` and `package-lock.json`
- Run `npm install --force` again

**Node.js not found:**
- Verify Node.js installation: `node --version`
- Ensure Node.js is added to your system PATH
